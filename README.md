# object-detection-project

> [!IMPORTANT]
> Work is in progress. Things will change

This is a project for the university subject: **Multimedia Services and Applications** (Usługi i aplikacje multimedialne)

## Project Title

**Object Detection in Video Data Stream** (Wykrywanie obiektów w strumieniu danych video)

## Project Structure

This project is divided into 2 main sections:

1. **/docs** - Contains all project documentation including partial reports (raport częściowy), final documentation (dokumentacja końcowa), and seminar presentation
2. **/src** - Contains the practical implementation to demonstrate real-time object detection scenarios

## Language

The project documentation and code are primarily in English (with some Polish documentation in the `/docs` section).
